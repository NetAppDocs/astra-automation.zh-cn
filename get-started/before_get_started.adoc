请务必查看 link:../rest-core/rest_implementation.html["核心 REST 实施"] 有关 REST 概念的信息以及有关如何设计 Astra REST API 的详细信息。



===== 拥有 Cloud Central 帐户凭据

您需要使用 * Auth0* 帐户凭据登录到 Astra Web 用户界面并生成 API 令牌。
