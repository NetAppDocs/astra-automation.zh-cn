== API resources

API 资源提供了多种可提供诊断信息和支持的 Astra 功能。

[cols="20,80"]
|===
| Type | Description 


| 事件 | 在 Astra 处理过程中记录的系统活动。 


| 通知 | 被视为足够重要的一小部分事件，可提供给用户。 


| 未读通知 | 用户尚未读取或检索的通知。 
|===
